# 🎮 Anto's Game Engine

A lightweight 3D/2D game engine project written in C++, built on top of **SFML**.

> ⚠️ This project is a personal learning project and is still in development.

---

## 🚀 Features

* Basic rendering system (via SFML)
* Custom game loop
* Object / entity handling
* Camera system (work in progress)
* Math utilities (vectors, transformations)

---

## 🧱 Project Structure

```
Anto-s-Game-Engine/
├── src/            # Source code
├── include/        # Headers
├── assets/         # Game assets (if any)
├── build/          # Build output (ignored)
└── ...
```

> The project follows a classic C++ separation between headers and implementation files.

---

## ⚙️ Getting Started

### Prerequisites

* C++ compiler (**MinGW recommended**)
* SFML
* CMake (recommended)

### Build

```bash
git clone https://github.com/Anto-1-12/Anto-s-Game-Engine.git
cd Anto-s-Game-Engine
mkdir build
cd build
cmake ..
cmake --build .
```

> 💡 This project is primarily developed and tested using **MinGW** rather than MSVC.
